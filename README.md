
# Carbon Footprint Prediction using Machine Learning

## 📌 Project Overview

This project predicts carbon footprint and classifies impact levels (High, Medium, Low) using machine learning techniques.

## 🎯 Objectives

* Predict carbon footprint value
* Classify environmental impact
* Identify important contributing factors

## 📊 Dataset

* Personal Carbon Footprint Behavior dataset
* 1400 records
* Features include distance, electricity usage, food type, screen time, etc.

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn

## 🤖 Models Used

* Random Forest (Classification)
* Linear Regression (Regression)

## 📈 Results

* Accuracy: ~93%
* Cross Validation: ~94%
* R² Score: ~0.95

## 📌 Conclusion

The model performs well and provides accurate predictions. It helps in understanding and reducing carbon footprint.
